# qf-infra-spring-boot-starter
微服务基础设施 强的飞起


# 功能特性

## 🚨 统一异常处理
- **全局异常捕获** - 自动拦截和处理控制器层异常
- **统一错误格式** - 标准化的错误响应结构

## 🌐 统一Web响应
- **响应标准化** - 统一的API响应格式
- **状态码封装** - HTTP状态码与业务状态码分离
- **泛型支持** - 类型安全的响应数据
- **分页封装** - 统一的分页数据结构

## 🚀 RocketMQ公共服务
- **消息生产** - 简化的消息发送接口
- **消息消费** - 注解驱动的消息监听
- **消息补偿** - 统一处理
- **消息追踪** - 完整的消息链路追踪
